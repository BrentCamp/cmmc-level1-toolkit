# CMMC Level 1 Compliance Toolkit

A practical reference toolkit for small businesses and DoD contractors 
preparing for CMMC Level 1 / FAR 52.204-21 compliance. Built from 
hands-on advisory work with DoD contractors at the Del Mar College 
Cyber Center.

This toolkit is designed to make compliance approachable for 
organizations without dedicated security staff. Every document is 
written in plain language with real implementation examples and 
evidence guidance.

## What Is CMMC Level 1?

CMMC Level 1 is the baseline cybersecurity standard required for any 
organization handling Federal Contract Information (FCI) as part of a 
DoD contract. It maps directly to the 15 practices in FAR 52.204-21 
and covers fundamental areas including access control, identification 
and authentication, physical protection, media disposal, system 
integrity, and boundary protection.

## About

This toolkit was developed by Brent Camp based on practical CMMC Level 
1 advisory work advising DoD contractors on compliance readiness through 
the Del Mar College Cyber Center.

It is intended as a practical reference and starting point for 
compliance preparation, not as legal or official compliance guidance. 
Always consult official CMMC documentation and a qualified assessor 
before submitting a self-assessment.
